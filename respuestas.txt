1.- ¿Qué es RVM (Ruby Version Manager)?
    Es una herramienta que nos permite tener tantos intérpretes de Ruby como queramos y además podemos definir un gemset (conjunto de gemas) para cada proyecto.

2.- ¿Cuáles son las diferencias entre la ejecución del programa "Hola Mundo" con el intérprete de Ruby (ruby helloWorld.rb) y el intérprete interactivo del ejercicio anterior?
    Cuando se ejecuta el programa "Hola Mundo" con el intérprete de Ruby se devuelve sólo el resultado. Cuando se ejecuta el programa "Hola Mundo" con el intérprete interactivo se devuelve la línea ejecutada y el resultado.

3.- ¿Cuáles son las diferencias entre la ejecución del programa "Hola Mundo" con el intérprete interactivo y con PRY?
    Cuando se ejecuta el programa "Hola Mundo" con PRY podemos pararlo y mostrar el código, pudiendo añadir nuevo código e interactuar con las variables. Cuando se ejecuta el programa "Hola Mundo" con el intérprete interactivo se devuelve la línea ejecutada y el resultado.
